Teste de Software

Dentre outros objetivos do teste de software realiza a verificação das funcionalidades de um produto se atendem as especificações e os requisitos, além de previnir falhas no produto, com isso, garantindo que o sistema atinja o nível de qualidade esperado e faça realmente o que se propõe a fazer.

Ou seja, o teste de software é a verificação feita sobre um sistema ou parte dele para garantir que uma determinada entrada produza, sempre, uma saída esperada.

Neste contexto temos duas formas de testes a manual e a de forma automática. Testes manuais são imprecindíveis para os projétos, pois visam principalmente cobrir os critériosde aceitação exigidos pelos usuários, clientes e demais partes interessadas. Apesar de parecerem relativamente fáceis, testes manuais devem ser realizados com objetivos préviamente definidos, planejados e com técnicas adequadas.

Já nos testes automáticos, basicamente, é a execução atravez do computador, de atividades que geralmente são realizadas de forma manual(por pessoas) utilizando ferramentas específicas para este fim. Isso é feito normalmente através da criação e execução dos chamados Scripts de teste.
